READ ME


1. Install Wamp, following the instructions on the website below and then run Wamp
	http://www.wampserver.com/en/
	
2. Download the GIT repository (https://github.com/daniellelambert/shoppingPlanner)

3.Source the database
	a. Go to localhost/phpmyadmin
	b. Click on import database
	c. Choose the file 'mealPlanner.sql'
	d. Click 'Go'
	
4. Move your code to C:\whatever\directories\wamp\www\shoppingPlan
	You will need to make a new folder called shoppingPlan
	
5. Restart the Wamp server

6. go to localhost/shoppingPlan and enjoy our website!
